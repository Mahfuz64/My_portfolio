# Portfolio (Single-page, Dark theme)
This is a minimal single-page portfolio site ready for GitHub Pages.

## Files
- index.html — main single-page site
- style.css — dark theme styles
- images/profile.jpg — placeholder profile image
- resume.pdf — placeholder resume (replace with your real CV)

## How to deploy on GitHub Pages
1. Create a new GitHub repository (e.g., `portfolio`).
2. Upload these files to the repository root (or push via git).
3. In the repository settings, enable GitHub Pages from the `main` branch and root folder.
4. Visit `https://yourusername.github.io/yourrepo/` after a minute or two.

Replace profile.jpg and resume.pdf with your real images and CV. Enjoy!
